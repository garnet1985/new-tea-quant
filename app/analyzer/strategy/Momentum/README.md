# Momentum策略

## 核心思想

基于均线动量的定期调仓策略，用于提供涨势中的收益基线。

## 策略逻辑

1. **买入时机**：每个周期（月/季度/年）的第一天
2. **动量计算**：`momentum = (MA短期 - MA长期) / MA长期`
   - 只计算上行趋势（MA短期 > MA长期）
3. **卖出时机**：周期最后一天自动卖出
4. **筛选机制**：后处理时按动能在同一天买入的股票中筛选前10%

## 配置说明

### 核心参数
- `short_ma`: 短期均线天数（默认20）
- `long_ma`: 长期均线天数（默认60）
- `rebalance_period`: 调仓周期（monthly/quarterly/yearly）
- `top_percentile`: 筛选百分比（默认0.10，即前10%）
