## 🎯 策略核心

### 信号特征（买入条件）

```python
1. ma_std < 0.03              # 均线收敛
2. position < 0.30            # 价格在震荡区低位
3. -0.176 < ma60slope < 0.176 # 趋势平稳（±10度）
```

### 目标设定

```python
分段目标（半年窗口）:
- 第一档: 10% （验证达成率：30.2%）
- 第二档: 20% （验证达成率：23.8%）
- 第三档: 30% （验证达成率：1.8%）

止损规则:
- 固定止损：-15%
- 保本止损：盈利10%后启动
```

## 🤖 机器学习成果

### 训练数据
- 样本数：4,052个（去重后）
- 特征数：22个
- 股票数：168只（随机采样）

### 模型表现（30%目标）

| 指标 | 数值 |
|-----|------|
| 测试集准确率 | 87.67% |
| AUC | 0.9490 |
| 精确率 | 85.57% |
| 召回率 | 82.64% |

### Top 5 关键特征

1. **drawdown_60d (25.5%)** - 前期60天跌幅
2. **ma60slope (21.0%)** - 趋势斜率
3. **volatility_60d (6.1%)** - 60天波动率
4. **amplitude_20d (5.8%)** - 震荡幅度
5. **position_60d (3.3%)** - 价格位置

**关键发现**：前期必须有足够跌幅（这是你之前没考虑的！）

## 🚀 下一步建议

### 优先级1：优化止损策略

```python
当前：-15%固定止损
问题：37.6%的信号触发止损

改进方向：
1. 收紧止损：-10%
2. 快速止损：3天内跌5% → 退出
3. 趋势确认：买入后5天必须涨，否则退出
```

### 优先级2：调整目标

```python
当前：30%目标
达成率：1.8%

建议：
- 改为15-20%目标
- 或动态目标（根据波动率）
```

### 优先级3：重新训练ML

```python
用真实交易收益重新标注
label = (真实交易收益 >= target)

而不是：
label = (潜在最大收益 >= target)
```
