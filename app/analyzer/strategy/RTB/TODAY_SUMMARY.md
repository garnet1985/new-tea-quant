## 📊 核心发现

### ML发现的3个关键优化

1. **前期跌幅最重要**
   ```
   drawdown_120d >= 0.20（半年至少跌20%）
   
   区分度：88%（最高）
   重要性：25.5%（排名第1）
   效果：过滤"假反转"
   ```

2. **止损不宜过严**
   ```
   -15% → -20%
   
   效果：
   - 止损率：37.6% → 32.7%
   - 平均收益：1.42% → 2.76%（+94%）
   ```

3. **保本止损有害**
   ```
   取消保本止损
   
   效果：
   - 止盈30%占比：23.6% → 31.1%（+8%）
   - 平均收益提升
   - 避免在震荡中被震出来
   ```

## 💡 关键经验

### 1. 验证方法很重要

```
错误做法：
❌ 只看最高价（理想化）
❌ 不去重（虚高94.6%）
❌ 不考虑止损

正确做法：
✅ 真实交易模拟（含止损止盈）
✅ 信号去重（冷却期120天）
✅ 多股票验证
```

### 2. ML的真正价值

```
不是：
❌ 直接提升30%达成率（反而下降）
❌ 神奇地解决所有问题

而是：
✅ 发现关键特征（drawdown_120d）
✅ 找出最优参数（止损-20%）
✅ 揭示问题（保本止损有害）
✅ 平均收益提升94%
```

### 3. 目标要现实

```
30%目标：太高（1.1%达成率）
20%目标：合理（31.7%达成率）⭐
15%目标：容易（36.0%达成率）

建议主目标：20%
```
