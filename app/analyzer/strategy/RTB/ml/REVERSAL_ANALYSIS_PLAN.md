# 反转点识别与RTB策略优化计划

## 📋 项目目标
基于历史数据识别重大反转点，通过机器学习分析找到RTB策略的最优参数组合，提升投资收益率。

## 🎯 核心思路

### 1. 反转点识别策略
**问题发现**：最初的方法识别出103个反转点，过于密集，不符合"重大反转"的定义。

**优化思路**：采用**分层识别**方法
- **第一层**：月线级别识别大波谷（过滤小波动）
- **第二层**：周线级别精确定位反转点（在月线谷底附近）

### 2. 数据驱动优化流程
```
平安银行验证 → 全样本应用 → 机器学习分析 → 参数优化
```

## 📊 已完成工作

### 阶段1：平安银行反转点识别验证
- ✅ 创建了严格版反转点识别脚本 `find_major_reversals.py`
- ✅ 识别出23个重大反转点（比之前的103个大幅减少）
- ✅ 分析了反转点特征：
  - 平均收益：29.5%
  - 平均持续时间：33.4周
  - 典型特征：价格位置极低、前期跌幅>50%、RSI超卖、均线收敛

### 阶段2：反转点时间分析
**发现的问题**：反转点仍然过于密集
- 2022年熊市底部：5个反转点（9-11月）
- 2023-2024年：18个反转点（10月-3月，几乎每周都有）

**结论**：需要进一步优化识别算法

## 🚀 下一步计划

### 阶段3：月线+周线分层识别（当前任务）
**目标**：实现更科学的分层识别方法

**具体步骤**：
1. **月线级别识别大波谷**
   - 使用月线数据识别明显的波谷
   - 设置合理的波谷深度阈值（如从峰值下跌>30%）
   - 设置波谷间隔阈值（避免过于密集）

2. **周线级别精确定位**
   - 在月线波谷附近（±2-3个月）寻找周线级别的精确反转点
   - 应用严格的技术指标条件
   - 确保反转点数量控制在10个左右

### 阶段4：全样本应用
**目标**：将验证过的算法应用到所有股票样本

**具体步骤**：
1. 选择代表性股票样本（如500只股票）
2. 应用分层识别算法
3. 收集所有反转点数据
4. 统计分析反转点特征分布

### 阶段5：机器学习分析
**目标**：基于大量反转点数据找到最优参数组合

**具体步骤**：
1. **特征工程**
   - 提取反转点的技术指标特征
   - 包括：价格位置、前期跌幅、均线收敛、RSI、波动性等

2. **参数优化**
   - 使用机器学习方法分析特征与收益的关系
   - 找到最优的参数阈值组合
   - 验证参数的有效性

3. **RTB策略优化**
   - 将优化后的参数应用到RTB策略
   - 回测验证策略效果
   - 对比优化前后的表现
