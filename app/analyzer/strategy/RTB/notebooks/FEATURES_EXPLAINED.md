# RTB 策略特征详解

## 🎯 你的策略目标

**找到反转信号**：股票从下跌/横盘进入上涨前的那个时刻

**信号特征**：
1. 下降趋势结束
2. 进入平稳震荡
3. 4条均线靠拢
4. 价格在低位

---

## 📊 特征清单

### 类别A：原始数据（直接从数据库读取）

| 特征 | 含义 | 示例 |
|-----|------|------|
| `date` | 日期 | 2024-10-11 |
| `open` | 开盘价 | 10.50 |
| `close` | 收盘价 | 10.00 |
| `highest` | 最高价 | 10.80 |
| `lowest` | 最低价 | 9.90 |
| `volume` | 成交量 | 1000000 |

**这些是基础数据，不需要计算**

---

### 类别B：均线（简单计算）

#### B1. MA5（5日均线）
```python
ma5 = 最近5天收盘价的平均值
```

**物理含义**：
- 代表最近1周的平均价格水平
- 反应短期趋势

**示例计算**：
```
假设最近5天收盘价：[9.80, 9.90, 10.00, 10.10, 10.20]
ma5 = (9.80 + 9.90 + 10.00 + 10.10 + 10.20) / 5 = 10.00
```

#### B2. MA10（10日均线）
```python
ma10 = 最近10天收盘价的平均值
```

**物理含义**：
- 代表最近2周的平均价格
- 比MA5更平滑，噪音更小

#### B3. MA20（20日均线）
```python
ma20 = 最近20天收盘价的平均值
```

**物理含义**：
- 代表最近1个月的平均价格
- 常用来判断中期趋势

#### B4. MA60（60日均线）
```python
ma60 = 最近60天收盘价的平均值
```

**物理含义**：
- 代表最近3个月的平均价格
- 长期趋势线

### 类别C：核心特征（需要深入讨论）

#### C1. ma_std（均线收敛度）⭐⭐⭐

**公式**：
```python
ma_std = std([ma5, ma10, ma20, ma60]) / close
```

**详细计算步骤**：
```python
1. 取4条均线的值：[ma5, ma10, ma20, ma60]
2. 计算标准差：std([...])
3. 除以当前价格（归一化）：/ close
```

**物理含义**：
- 衡量4条均线之间的"离散程度"
- 值越小 → 均线越靠近 → 越"收敛"
- 值越大 → 均线越分散 → 越"发散"

**为什么需要这个特征？**
```
你的策略说：4条均线要"足够靠近"
↓
那怎么定义"足够靠近"？
↓
用标准差来量化！
```

**具体案例**：

**案例1：收敛状态（ma_std = 0.01）**
```
价格 = 10.00
ma5  = 10.05  ← 只差 0.05
ma10 = 10.03  ← 只差 0.03
ma20 = 10.01  ← 只差 0.01
ma60 = 9.96   ← 只差 0.04

std([10.05, 10.03, 10.01, 9.96]) = 0.037
ma_std = 0.037 / 10.00 = 0.0037 ≈ 0.004 (0.4%)

解读：4条均线几乎重叠！
```

**案例2：发散状态（ma_std = 0.05）**
```
价格 = 10.00
ma5  = 11.00  ← 差 1.00
ma10 = 10.50  ← 差 0.50
ma20 = 10.00  ← 差 0.00
ma60 = 9.00   ← 差 1.00

std([11.00, 10.50, 10.00, 9.00]) = 0.816
ma_std = 0.816 / 10.00 = 0.0816 ≈ 0.08 (8%)

解读：4条均线距离很远，正在多头/空头排列
```

**💡 你关心的问题：**
> "4个均线可能很靠近，但都离真实价格很远"

**回答：你说得对！所以需要 C2 特征来补充！**

---

#### C2. distance_to_ma_center（价格距离均线中心）⭐⭐⭐

**公式**：
```python
ma_center = mean([ma5, ma10, ma20, ma60])
distance_to_ma_center = abs(close - ma_center) / close
```

**物理含义**：
- 价格离4条均线的"中心点"有多远
- 值越小 → 价格越接近均线束
- 值越大 → 价格偏离均线束

**为什么需要这个特征？**
```
ma_std 只能判断"均线彼此靠近"
但不能判断"价格是否也在这个区域"

例如：
- 均线都在12元附近收敛
- 但价格在10元
- 这不是震荡，这是下跌后的反弹受阻！

所以需要 distance_to_ma_center 来约束
```

**具体案例**：

**案例1：价格在均线中心（好信号）**
```
价格 = 10.00
ma5  = 10.05
ma10 = 10.03
ma20 = 9.98
ma60 = 9.94

ma_center = (10.05 + 10.03 + 9.98 + 9.94) / 4 = 10.00
distance = |10.00 - 10.00| / 10.00 = 0.00

解读：价格正好在均线中心，真正的震荡！
```

**案例2：价格远离均线中心（坏信号）**
```
价格 = 10.00
ma5  = 11.95
ma10 = 12.03
ma20 = 12.05
ma60 = 11.97

ma_center = (11.95 + 12.03 + 12.05 + 11.97) / 4 = 12.00
distance = |10.00 - 12.00| / 10.00 = 0.20 (20%)

解读：均线虽然收敛，但都在价格上方20%！
这是空头排列压制，不是震荡！
```

#### C3. price_position（价格在震荡区间的位置）⭐⭐

**公式**：
```python
high_20d = 最近20天的最高价
low_20d = 最近20天的最低价
price_position = (close - low_20d) / (high_20d - low_20d)
```

**物理含义**：
- 0 = 价格在震荡区间底部
- 1 = 价格在震荡区间顶部
- 0.5 = 价格在震荡区间中间

**为什么需要这个特征？**
```
你的策略说："价格在平稳区的下方"
↓
怎么定义"下方"？
↓
把震荡区间归一化到 0-1
然后设定阈值，如 < 0.33 就是下1/3
```

**具体案例**：

```
最近20天价格范围：9.00 - 12.00

情况1：今天价格 = 9.30
price_position = (9.30 - 9.00) / (12.00 - 9.00) = 0.10
解读：在震荡区底部 10% 位置 ✅ 低位！

情况2：今天价格 = 11.50
price_position = (11.50 - 9.00) / (12.00 - 9.00) = 0.83
解读：在震荡区顶部 83% 位置 ❌ 太高了！
```

#### C4. volatility_20d（20日波动率）⭐

**公式**：
```python
returns = 每日收益率
volatility_20d = std(最近20天的returns)
```

**物理含义**：
- 价格波动的剧烈程度
- 值越大 → 波动越大 → 股票越"活跃"
- 值越小 → 波动越小 → 股票越"死"

**为什么需要这个特征？**
```
过滤掉"死股"：
- 如果波动率太小（如 0.01），说明股票不怎么动
- 即使收敛了，也可能只是横盘，不会有大涨幅
```

**具体案例**：

```
股票A：最近20天每天涨跌 ±0.5%
volatility = 0.005 (0.5%) ❌ 太死了，不考虑

股票B：最近20天每天涨跌 ±3%
volatility = 0.03 (3%) ✅ 够活跃，可能有30%涨幅
```

---

#### C5. trend_recent_20d（最近20天趋势）⭐

**公式**：
```python
trend_recent_20d = (close - close_20d_ago) / close_20d_ago
```

**物理含义**：
- 最近20天的涨跌幅
- 正值 = 上涨
- 负值 = 下跌
- 接近0 = 平稳

**为什么需要这个特征？**
```
你的策略说："下降趋势结束 → 平稳震荡"
↓
怎么判断是否平稳？
↓
看20天涨跌幅是否接近0
```

**具体案例**：

```
情况1：trend_recent_20d = -0.05 (-5%)
解读：最近20天跌了5%，趋势向下 ❌

情况2：trend_recent_20d = 0.01 (+1%)
解读：最近20天几乎没动，平稳 ✅

情况3：trend_recent_20d = 0.20 (+20%)
解读：最近20天涨了20%，已经在上涨了 ⚠️ 可能错过了
```
