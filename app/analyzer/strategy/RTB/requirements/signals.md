# RTB 策略信号定义 (V8 最终版)

## 策略概述

RTB (Rebound from Trough after Big drop) 策略旨在捕捉**股票从大幅下跌后的反转机会**。

核心思想：
- 股票经历了较大跌幅（120天跌幅≥20%）
- 进入平稳震荡阶段（4条均线收敛）
- 价格处于震荡区间下方（低位）
- 长期趋势开始微涨（MA60斜率为正）
- 近期有回调（买入前20天下跌）

## 入场信号条件

策略使用**6个过滤条件**组合，所有条件必须同时满足：

### 1. 均线与收盘价收敛度 (ma_std)

**定义**：
```python
ma_std = std([ma5, ma10, ma20, ma60]) / close
```

**阈值**：`ma_std < 0.03`

**含义**：
- 4条均线（5日、10日、20日、60日）相互靠近
- 除以收盘价进行归一化，便于不同价格股票对比
- 表示股票从趋势行情进入震荡行情

**验证结果**：
- 合格率：约30%的K线满足此条件
- 效果：有效筛选出震荡阶段

---

### 2. 价格相对位置 (position)

**定义**：
```python
high_20d = 最近20天最高价
low_20d = 最近20天最低价
position = (close - low_20d) / (high_20d - low_20d)
```

**阈值**：`position < 0.30`

**含义**：
- 当前价格在最近20天高低点区间的位置
- < 0.30 表示价格在下方30%区域
- 确保买在"低位"，而不是高位

**验证结果**：
- 合格率：约25%的K线满足
- 效果：有效筛选低位买入点

---

### 3. 前期跌幅 (drawdown_120d) ⭐ 最重要

**定义**：
```python
high_120d = 最近120天最高价
drawdown_120d = (high_120d - close) / high_120d
```

**阈值**：`drawdown_120d >= 0.20`

**含义**：
- 股票在最近120天（约半年）从高点下跌至少20%
- 这是ML识别的**最关键特征**
- 确保是"大跌后反弹"而不是"横盘突破"

**验证结果**：
- Feature Importance: 0.35（最高）
- 成功交易平均 drawdown_120d: 0.396
- 失败交易平均 drawdown_120d: 0.316
- 差异显著（T-test p < 0.001）

---

### 4. 长期趋势斜率 (ma60slope)

**定义**：
```python
ma60_20d_ago = 20天前的MA60
ma60slope = (ma60 - ma60_20d_ago) / ma60_20d_ago
```

**阈值**：`0.009 < ma60slope < 0.176`

**含义**：
- MA60的20天变化率
- 下限 0.009：趋势必须微涨（不能平稳或下跌）
- 上限 0.176：约等于±10度角，避免过于陡峭
- 确保长期趋势开始转好

**验证结果**：
- 成功交易平均 ma60slope: 0.045（微涨）
- 失败交易平均 ma60slope: 0.021（接近平）
- ML识别：必须微涨才是真反转

---

### 5. 近期回调 (return_20d)

**定义**：
```python
return_20d = (close - close_20d_ago) / close_20d_ago
```

**阈值**：`return_20d <= -0.07`

**含义**：
- 买入前20天的涨跌幅
- <= -0.07 表示近期下跌至少7%
- 确保是"下跌后的反转"而不是"持续上涨"

**验证结果**：
- 成功交易平均 return_20d: -0.142
- 失败交易平均 return_20d: -0.109
- 买入前跌得更多的机会更好

---

### 6. 价格相对MA20位置 (close_to_ma20)

**定义**：
```python
close_to_ma20 = (close - ma20) / ma20
```

**阈值**：`close_to_ma20 <= -0.05`

**含义**：
- 当前价格相对MA20的距离
- <= -0.05 表示价格低于MA20至少5%
- 确保买在均线下方（低位）

**验证结果**：
- 成功交易平均 close_to_ma20: -0.087
- 失败交易平均 close_to_ma20: -0.069
- 价格越低于MA20，机会越好

---

## 信号组合逻辑

所有6个条件采用**AND逻辑**（必须同时满足）：

```python
signal = (
    ma_std < 0.03 AND
    position < 0.30 AND
    drawdown_120d >= 0.20 AND
    0.009 < ma60slope < 0.176 AND
    return_20d <= -0.07 AND
    close_to_ma20 <= -0.05
)
```

## 信号去重

为避免重复计数同一机会，采用**120天冷却期**：
- 同一只股票触发信号后
- 120天内不再触发新信号
- 确保每个信号代表独立的投资机会

## 验证结果（50只股票，2010-2024）

|-----|------|
| 去重后信号数 | 517 |
| 信号频率 | 约10个/年 |
| 平均持有天数 | 50-80天 |
