# RTB 策略 - 信号定义文档

## 📋 文档说明

这个文档定义 RTB 策略的**入场信号**（买入时机）

- **特征**：用什么指标来判断
- **阈值**：每个指标的临界值（可以先凭经验设置）
- **组合逻辑**：特征如何组合（AND/OR）

---

## 🎯 核心信号逻辑

### 信号含义
反转信号：股票从下跌/横盘即将进入上涨的时刻

### 信号条件（用自然语言描述）
1. 下降趋势已经结束
2. 进入平稳震荡状态
3. 4条均线靠拢收敛
4. 价格处于震荡区的低位

---

## 📊 特征定义

### 特征1: 均线与收盘价收敛度（ma_std）

**计算方式**：
```python
ma_std = std([ma5, ma10, ma20, ma60, close])
```

**物理含义**：
- 衡量4条均线还有收盘价之间的离散程度
- 值越小 = 越收敛

**初步阈值**：
```python
ma_std < 0.03  # 3%以内算收敛
```

**✅ 验证结果**：
- 实际信号的 ma_std 平均值：0.017-0.019
- 说明阈值合理，大部分信号都在收敛状态

**可调整空间**：
可以测试 0.01 ~ 0.05 之间

---

### 特征2: 价格相对均线位置（distance_to_ma_center）

**计算方式**：
```python
position = (close - 震荡区间min) / (震荡区间max - 震荡区间min)
```

**物理含义**：
- 收盘价处于当前震荡趋势的下方
- 值越小 = 价格越靠近合理的买入点，

**初步阈值**：
```python
position < 0.30  # 在震荡区下30%位置
```

**✅ 验证结果**：
- 实际信号的 position 平均值：0.15-0.16
- 说明大部分信号都在震荡区底部（很好的买入位置）

**为什么选这个阈值**：
买入点越低可承受的止损可以更大

**可调整空间**：
可以测试 0.10 ~ 0.50 之间

---


### 特征3: 均线斜率

**计算方式**：
```python
-0.5 < ma60slope < 0.5
```

**物理含义**：
- 我们的投资不是出于一个强下跌过程中，避免不是反转而是反弹
- 我们的投资不是出于一个强上涨过程中，避免买在山顶

**初步阈值**：
```python
ma60slope = (ma60 - ma60_20d_ago) / ma60_20d_ago
-0.176 < ma60slope < 0.176  # 对应±10度，即20天内涨跌幅在±17.6%以内
```

**✅ 验证结果**：
- 实际信号的 ma60slope 平均值：0.003-0.005
- 说明大部分信号都在很平稳的趋势中

**为什么选这个阈值**：
- 避免在强下跌过程中买入（不是反转而是反弹）
- 避免在强上涨过程中买入（可能买在山顶）
- ±10度确保趋势平缓

**可调整空间**：
可以测试 ±5度(-0.09) ~ ±15度(-0.27) 之间

---

## 🔗 特征组合逻辑

### ✅ 最终方案：全部满足（AND）

```python
信号条件 = (
    ma_std < 0.03 AND
    position < 0.30 AND
    -0.176 < ma60slope < 0.176
)
```

**验证结果**：
- 信号占比：12-16%（不会太频繁）
- 信号质量：高（平均收益27.89%）

**严格筛选，信号较少但质量高** ✅ 已验证



## 📝 其他考虑

### 需要排除的情况

1. **新股/次新股**：上市不足1年
2. **ST股票**：已经被过滤
3. **停牌股票**：无法交易
4. **涨停/跌停**：极端情况

### 时间周期

- **基础数据**：日线
- **均线计算**：5, 10, 20, 60日
- **震荡区间**：20日
- **波动率**：20日
