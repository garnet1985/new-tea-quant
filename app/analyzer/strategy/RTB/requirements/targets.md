# RTB 策略目标定义 (V8 最终版)

## 止盈目标

策略采用**分段止盈**策略，在不同盈利水平分批卖出，平衡收益和风险。

### V8 最终方案（优化后）

| 盈利水平 | 平仓比例 | 剩余仓位 | 收益贡献 |
|---------|---------|---------|---------|
| 10% | 0% | 100% | 0% |
| 20% | 50% | 50% | 10% |
| 30% | 50% | 0% | 15% |
| **总计** | | | **25%** |

**计算示例**（涨到30%）：
```
10%档: 0%  × 10% = 0.00%
20%档: 50% × 20% = 10.00%
30%档: 50% × 30% = 15.00%
─────────────────────────
总收益: 25.00%
```

### 为什么选择 (0/50/50)？

基于数学期望值优化：

1. **10%档不平仓（0%）**
   - 10%只是起点，不是目标
   - 继续持有等待更高收益
   - 10-20%保持率：80.8%（大多数能到20%）

2. **20%档锁定一半（50%）** ⭐ 核心
   - 20-30%区间是主要收益来源（32.9%的交易）
   - 锁定50%保护利润
   - 剩余50%继续搏高收益
   - 平衡风险和收益的最佳点

3. **30%档全部止盈（50%）**
   - 到30%全部离场
   - 20-30%保持率：1.1%（很难到30%）
   - 不再搏更高收益（达成率太低）

### 方案对比（历史验证）

| 方案 | 10%档 | 20%档 | 30%档 | 期望值 | 胜率 | 20%达成 | 30%达成 |
|-----|-------|-------|-------|--------|------|---------|---------|
| V7 (30/30/40) | 30% | 30% | 40% | 2.85% | 54.7% | 33.8% | 1.0% |
| **V8 (0/50/50)** | **0%** | **50%** | **50%** | **3.23%** | **52.0%** | **35.2%** | **9.3%** |
| 全仓 (0/0/100) | 0% | 0% | 100% | 3.89% | 48.2% | 34.4% | 0.4% |

**结论**：
- V8期望值比V7高13.3%
- V8保持52%胜率（过半）
- V8的30%达成率是V7的9倍（因为剩余50%仓位能吃到更多涨幅）

## 止损策略

### 固定止损：-20%

```python
if current_return <= -0.20:
    止损离场
```

**为什么选择-20%？**

基于ML分析结果：
- -15%止损：期望值 2.76%
- -20%止损：期望值 2.85% ✅ 最优
- -25%止损：期望值 2.71%

**原因**：
- 止损过严（-15%）：增加止损次数，错失反弹
- 止损适中（-20%）：平衡保护和容忍度
- 止损过松（-25%）：亏损加大，期望值下降

### 取消保本止损

**V7曾使用保本止损**：
- 盈利10%后，跌回成本价触发止损

**V8取消原因**：
- 分析显示保本止损**降低期望值**
- 震荡市场中频繁触发
- 失去后续上涨机会
- 数据支持：取消后期望值提升

## 持有期限

**最大持有期：120天（约4个月）**

未触发止盈或止损时：
- 持有到120天自动平仓
- 按实际收益结算

**数据**：
- 35%的交易到期（120天）
- 到期交易平均收益：2.80%
- 大部分盈利交易在60天内完成

## 数学期望值

### 期望值计算

```
期望值 = P(盈利) × 平均盈利 + P(亏损) × 平均亏损
       = 52.0% × 21.96% + 47.8% × (-17.15%)
       = 11.42% + (-8.20%)
       = 3.22%
```

**扣除交易成本（0.3%）**：
```
净期望值 = 3.22% - 0.3% = 2.92%
```

### 盈亏比

```
盈亏比 = 平均盈利 / |平均亏损|
       = 21.96% / 17.15%
       = 1.28
```

**解读**：
- 盈亏比 > 1：平均盈利大于平均亏损 ✅
- 配合52%胜率，形成正期望值

### 凯利准则

```
凯利公式: f = (p×b - q) / b

p (胜率) = 52.0%
q (败率) = 48.0%
b (盈亏比) = 1.28

凯利值 = (0.52 × 1.28 - 0.48) / 1.28
       = 0.146
       = 14.6%
```

**建议仓位**：每次投资使用14.6%的资金
- 保守投资者可减半至7-8%

## 适用场景

### 适合的市场环境

✅ 熊市底部（大跌后反弹）
✅ 震荡市（均线收敛）
✅ 个股机会（非系统性下跌）

### 不适合的市场环境

❌ 强势上涨（不满足drawdown条件）
❌ 持续下跌（反弹无力）
❌ 系统性风险（如金融危机）
