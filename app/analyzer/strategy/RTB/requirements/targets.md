# RTB 策略 - 目标定义文档

## 📋 文档说明

这个文档定义 RTB 策略的**盈利目标**和**评估标准**

- **收益率目标**：期望赚多少
- **时间窗口**：多久内达到
- **止盈止损**：什么时候卖出
- **评估标准**：怎么算成功

## 💰 收益率目标

### 方案A：单一固定目标

**目标收益率**：
```python
target_return = 30%  # 未来N天内最高涨幅
```

**时间窗口**：
```python
time_window = 20天  # 约1个月
```

**评估方式**：
```python
成功 = 未来20天内最高价涨幅 >= 30%
```

---

### 方案B：分段目标

**第一档（保守）**：
```python
target_return_1 = 10%
time_window_1 = 半年内（120天）
```

✅ **验证结果**：95.1% 达成率（几乎必达）

**第二档（合理）**：
```python
target_return_2 = 20%
time_window_2 = 半年内（120天）
```

✅ **验证结果**：65.7% 达成率（大概率达成）

**第三档（激进）**：
```python
target_return_3 = 30%
time_window_3 = 半年内（120天）
```

✅ **验证结果**：34.7% 达成率（三分之一能达成）

**第四档（超级）**：
```python
target_return_4 = 动态止损，上限不限
time_window_4 = 半年内（120天）
```

✅ **验证结果**：平均收益 27.89%，最高收益 231.53%


**评估方式**：
```python
优秀 = 达到第三档（30%+）
良好 = 达到第二档（20%+）
及格 = 达到第一档（10%+）
失败 = 三个都没达到
```

```python
if volatility_20d > 0.05:  # 高波动股

elif volatility_20d > 0.03:  # 中等波动
    target_return = 25%
    time_window = 20天
    
else:  # 低波动股
    target_return = 15%
    time_window = 15天
```

**评估方式**：
```python
成功 = 未来time_window天内达到对应的target_return
```

## 📊 收益计算方式

### 选项1：最大收益率（推荐用于验证）

```python
future_max_return = (未来N天最高价 - 买入价) / 买入价
```

**优点**：
- 评估信号的"潜力"
- 不考虑具体买卖时机
- 适合初期验证

**缺点**：
- 实际可能拿不到最高点
- 过于理想化

---

### 选项2：固定持有收益率

```python
future_return = (未来第N天收盘价 - 买入价) / 买入价
```

**优点**：
- 更真实（持有到期）
- 简单明确

**缺点**：
- 可能错过中间高点
- 到期可能正好是低点

---

### 选项3：止盈止损后收益率（最真实）

```python
考虑实际交易规则：
- 达到止盈目标 → 卖出
- 触发止损 → 卖出
- 到期 → 卖出

actual_return = (卖出价 - 买入价) / 买入价
```

**优点**：
- 最接近实际交易
- 考虑了风险控制

**缺点**：
- 需要定义止盈止损规则
- 计算复杂
