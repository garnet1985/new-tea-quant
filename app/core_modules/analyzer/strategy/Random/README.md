# Random策略

## 📋 策略概述

Random策略是一个有趣的实验性策略，用于测试随机投资的效果。

### 🎯 核心思想

1. **随机投资**: 5%的概率随机掷骰子决定是否投资
2. **动态止损**: 止损基于投资前20天的累积振幅delta
3. **固定止盈**: 止盈永远是止损的1.5倍

### 🔧 策略参数

- **投资概率**: 5% (0.05)
- **振幅回望天数**: 20天
- **止盈止损比例**: 1.5倍
- **止损范围**: -5% 到 -20%
- **止盈范围**: 7.5% 到 30%

### 📊 策略逻辑

#### 投资决策
- 随机生成0-1之间的数字
- 如果数字 ≤ 0.05，则进行投资
- 否则跳过该股票

#### 止损计算
- 计算投资前20天的日振幅标准差
- 止损比例 = max(-0.20, min(-0.05, -振幅delta))
- 确保止损在-5%到-20%之间

#### 止盈计算
- 止盈比例 = |止损比例| × 1.5
- 确保止盈是止损的1.5倍

### 🎲 使用场景

1. **基准测试**: 作为其他策略的基准对比
2. **随机性研究**: 研究随机投资的市场表现
3. **振幅研究**: 研究振幅delta对投资效果的影响

### 📈 预期表现

由于是随机投资，预期表现应该接近市场平均水平，但通过动态止损止盈机制，可能会获得一些超额收益。

### 🔍 实验价值

- 验证随机投资的有效性
- 测试动态止损止盈机制
- 为其他策略提供基准对比
