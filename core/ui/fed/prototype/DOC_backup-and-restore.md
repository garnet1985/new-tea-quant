# 页面文档：`backup-and-restore.html`

## 这个页面是干什么的？
备份与恢复页，承载用户可操作的数据安全能力（备份、恢复）。

## 页面目的与用户价值
- 给用户可见且可控的数据保护入口，降低误操作风险。
- 用清晰分栏减少“备份”和“恢复”流程混淆。

## 页面功能描述
- 使用 tab 分离两个独立 block：
  - 备份
  - 恢复
- 备份支持模式选择：
  - 整体备份
  - 单表备份（仅此模式显示表下拉）
- 恢复支持备份版本选择与执行反馈。
- 两侧都有进度与结果提示区域。

## 页面假设与 Placeholder
- 当前不接真实备份引擎，进度和结果为模拟。
- 备份版本列表与表列表为静态样例。
- 权限校验、冲突处理、回滚机制暂未接入。
