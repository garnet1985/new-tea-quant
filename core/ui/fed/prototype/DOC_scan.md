# 页面文档：`scan.html`

## 这个页面是干什么的？
机会扫描页，用选中的已启用策略扫描市场机会并输出报告。

## 页面目的与用户价值
- 让用户从“单策略调试”进入“多策略批量扫描”。
- 快速得出当前市场机会分布，提升决策效率。
- 提供“调试策略”回流入口，形成闭环。

## 页面功能描述
- 策略 data grid（每行带复选框，仅 enabled 策略可选）。
- 扫描模式：
  - 严格模式：要求数据 up-to-date。
  - 扫描演示：使用最新可用交易日做演示扫描。
- 顶部“开始扫描”动作。
- 扫描进度展示（进度条 + 文本）。
- 扫描完成报告：
  - 每个策略的机会数量。
  - 点击机会数量弹窗查看机会明细（股票 ID、名称、建议最高买入价）。
- “验证”入口调整为“调试策略”。

## 页面假设与 Placeholder
- 扫描结果为 mock，不接真实任务编排系统。
- 报告统计和机会明细来自前端内置样例。
- 调试策略跳转仅做 UI 联动，不校验后端状态。
