# 页面文档：`workbench-detail.html`

## 这个页面是干什么的？
策略验证页（单次实验工作台），用于按三层执行计划运行并查看结果：枚举机会、价格回测、资金模拟。

## 页面目的与用户价值
- 让用户可视化理解“策略从机会到收益”的完整链路。
- 通过分层执行降低调试复杂度，定位策略问题更快。
- 在不离开页面的情况下完成策略验证、观察核心指标、查看样本股票。

## 页面功能描述
- 显示实验上下文（策略信息、回测数据区间等关键提示）。
- 三层执行计划：
  - 枚举机会：显示机会数量等指标。
  - 价格回测：显示胜率、ROI、平均投资时长。
  - 资金模拟：显示收益率/收益、资金起止、曲线占位。
- 依赖关系：资金层依赖枚举层；价格层不应清空已完成资金层状态。
- 结果区初始使用占位值（`--`），执行后填充。
- 当三层都完成后，显示样本股票列表说明区。
- 策略级动作按钮：保存、克隆、启用/禁用、删除。

## 页面假设与 Placeholder
- 执行过程与结果均为前端模拟（定时器/随机样例）。
- 曲线为 placeholder，不接入图表引擎。
- 保存/克隆/启用/删除仅做交互占位，不落库。
