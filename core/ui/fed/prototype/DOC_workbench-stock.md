# 页面文档：`workbench-stock.html`

## 这个页面是干什么的？
单股票 K 线分析页，承接策略验证页中的样本股票，按执行层展示图上标记意图。

## 页面目的与用户价值
- 将抽象回测结果落到具体标的上，帮助用户做“可解释性核对”。
- 通过 tab 分层，避免一次展示过多信息造成认知负担。
- 作为未来接入真实图表标注能力的 UI 骨架。

## 页面功能描述
- 顶部单行显示策略名、股票名及基本信息。
- 返回按钮：可回到上游策略验证页。
- 三个 tab：
  - 枚举机会：说明将标注机会点。
  - 价格回测：说明将标注 buy/sell 点。
  - 资金模拟：说明将标注买点与买入数量。
- 底部 K 线区域使用图表占位。

## 页面假设与 Placeholder
- 当前不接入真实行情与图表组件。
- tab 下仅显示“标注意图说明”文本，不做真实标注。
- 页面数据通过 URL 参数或前端 mock 提供。
