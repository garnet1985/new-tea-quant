API 文档 → HTML（带语义 class）

粘贴「API 文档」Markdown（含 ### 标题、描述、函数签名、参数 表、输出、Example；可选 状态：deprecated / 即将淘汰 会生成对应 class）。

生成的 HTML 使用的 class（便于在主题里写样式）
  • .api-doc — 根容器
  • .api-doc__section / .api-doc__section-title — 大节（如 DataSourceManager）
  • .api-entry — 单个 API 卡片
  • .api-entry__title-box — 标题外框（默认色；.api-entry--deprecated / .api-entry--warning 时为 warning/alert 色）
  • .api-entry__title — API 标题（无序号）
  • .api-entry__description / .api-entry__fn / .api-entry__output — 描述、函数签名、输出（标签与内容分两行）
  • .api-entry__params — 参数区；内含 .api-params-table 表格
  • .api-entry__example — 示例区；内含 .api-example__code 代码块
下方为预览，复制 HTML 后可粘贴到 CKEditor 或静态页。
